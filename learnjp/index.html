About

This is a very concise guide on how to read (VNs, LNs, manga etc.) in Japanese. For more comprehensive guides please see Kuri's guide and/or Shoui's guide. If you have any questions please feel free to contact me on Discord (username: nullsp_ce)

Foundation

Kana

Hiragana and Katakana are the 2 basic "alphabets" used in Japanese. You can learn them here.

Basic Vocabulary

Memorise ~1000 common words. I recommend you do this with Anki and the Kaishi 1.5k deck.

Basic Grammar

Read through one of these grammar guides (pick whichever you like the most, they teach the same content):

  • Sakubi Grammar Guide
  • Tae Kim's Grammar Guide
  • IMABI

Reading

Yomitan Setup

The Yomitan browser extension [Firefox] [Chrome] will allow you to look up unknown words as you hover over them. The most important dictionaries you should install on it are:

  • Jitendex: a JP-EN vocabulary dictionary
  • Dictionary of Japanese Grammar: a JP-EN grammar dictionary
  • JPDB Frequency v2.2: a frequency dictionary

Read!

Read whatever you like! If you need a recommendation, 月姫 (Tsukihime) is an easy visual novel that you can read for free in your browser. You can change the language to Japanese via Config > Advanced > Language > 日本語.

You can read light novels (as EPUB or HTMLZ files) with ッツ Reader.

You can read various manga (with Yomitan-able HTML text) at Mokuro.
